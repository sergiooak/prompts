# prompts

🇧🇷 [Leia em português](./README.pt-BR.md)

A personal, growing library of prompts I reuse often with AI coding tools (GitHub Copilot, Cursor, Claude Code). Public in case it's useful to someone else — use anything here however you like.

## Structure

```
prompts/
├── coding/
│   └── general.md      # reusable prompts, grouped by category
├── docs/
│   └── README.md        # philosophy, conventions, how to contribute (for humans)
├── agents/
│   └── README.md        # same info, written for AI agents
└── AGENTS.md             # entry point for AI agents
```

## Usage

Each prompt is one line: an emoji (visual anchor, not a search key) followed by a ready-to-run instruction. Open [`coding/general.md`](./coding/general.md), grab the one you need, paste it into your tool of choice.

For the reasoning behind the structure and how to add new prompts, see [`/docs`](./docs/README.md).
